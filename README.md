### 사용한 기술

- ☑️Java 11
- ☑️SpringBoot
- ☑️AWS EC2 :
    - 서버 배포를 위해 사용
- ☑️AWS RDS :
    - 데이터베이스 구축, 관리 (MYSQL)
- ☑️Spring Security :
    - 로그인 기능을 위해 사용
- ☑️Spring JPA : 
    - 관계형 데이터베이스 MAPPING
